# 🕒 Digital Clock using Python (Tkinter)

A simple real-time digital clock built using Python’s Tkinter library.  
It displays the current time and date and updates every second.

---

## 📌 Features

- Live digital time (updates every second)
- Displays current date
- Simple GUI using Tkinter
- Lightweight and beginner-friendly project

---

## 🛠️ Technologies Used

- Python 🐍
- Tkinter (GUI Library)
- Time module (strftime)

---















---

## 🚀 How to Run

1. Make sure Python is installed on your system
2. Download or clone the repository
3. Open terminal in project folder
4. Run the file:

```bash
python digital_clock.py
